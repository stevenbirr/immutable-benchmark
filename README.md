# Benchmark for Immutable.js operations

This script is a benchmark for testing speed performance of some [Immutable.js](https://github.com/facebook/immutable-js) functions using [Faker.js](https://github.com/Marak/faker.js) and [Benchmark.js](https://github.com/bestiejs/benchmark.js).

## Currently supported
* List.filter()
* List.some()

# Usage
```
npm install
npm start
```
